JavaScript for Beginners

What is JavaScript

  1. High level, interpreted programming language

    2. interpreted, is that no compiler is necessary

  2. Conforms to the ECMA Script specifications
  3. Multi-paradigm

    4. Can right object oriented or functional code.

  4. Runs on the client/browser as well as on the server [Node.js]

    5. Uses a runtime - Node.js to run certain features in the server side, like forms or database storage.

Why Learn JavaScript

  1. It is thee programming language of the browser
  2. Build very interactive user interfaces with frameworks like React
  3. Ised in building very fast server side and full stack applications
  4. ISed in mobile development[React Native, NativeScripte, Iconic]
  5. Used in desktop application development [Framework - Electron JS]

What we LEARN

  1. Variables & Data Types
  2. Arrays
  3. Object Literals
  4. MEthods for strings, arrays, objects, etc
  5. Loops - for, while, for.. of, forEach, map
  6. Conditionals [if, ternary & switch]
  7. Functions [normal & arrow]
  8. OOP [Prototype & classes]
  9. DOM Selection
  10. DOM manipulation
  11. Events
  12. Basic Form Validation

Further Learning

JS DOM, JS OOP, Higher Order Array Methods, JS Async, Fetch API & Ajax, Vanilla JS Playlists, JS Frameworks [React, Angular and Vue]
